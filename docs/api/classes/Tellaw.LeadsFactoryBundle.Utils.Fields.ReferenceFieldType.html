\Tellaw\LeadsFactoryBundle\Utils\Fields

\Tellaw\LeadsFactoryBundle\Utils\FieldsReferenceFieldType

Field of type ReferenceList will be identified by the type <b>reference-list</b> and will be used to display content of a reference list.

* selection : "multiple" or "single" will switch from Select field to checkbox field
display : "options" will force display of options instead of default selection

Exemple : <field type="reference-list" selection="single" display="options"/>
