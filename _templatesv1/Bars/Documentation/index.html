    How to

    1. How to change the value of New Order and New Sales widget?

    Go to the folder with the name of “js” which contains a file “ticker.js” and open it on your HTML editor. Here you will find the classes “.ticker--one” and “.ticker--two” where the former is meant for New Order and the latter contains the code for New Sales, as shown below: 

    	$(".ticker--one").ticker({

	speed: "100",

	end: "90"

	});

    Here the code speed: "100" is meant for adjusting the speed of the numbers (total number of orders) till they reach the end point. The code end: "90" depicts the end point where the numbers will stop increasing. So, you can set the speed and end point as you like. 

    $(".ticker--two").ticker({
  
    speed: "1",
    
    end: "1256"
  
  });

    Here the code speed: "1" is meant for adjusting the speed of the numbers (total number of orders) till they reach the end point. The code end: "1256" depicts the end point where the numbers will stop increasing. So, you can set the speed and end point as you like.

    2. How to change the color of Easing Pie Chart?

    Go to the folder with the name of “js” which contains a file “easy-pie-chart.js” and open it on your HTML editor. Here you will find the classes “.review” and “.earning” where the former is meant for Reviews and the latter contains the code for Earning.

    Changing the Color of Reviews (Easing Pie Chart): 
    Locate the code as shown below:

/*** Reviews Circle Stats ***/ 
$('.review').easyPieChart({
animate: 1000,
barColor: '#f2af32',
trackColor: '#e1e1e1',
scaleColor: '#cecece',
onStep: function(value) {
this.$el.find('span').text(~~value);
}
});

    Here you can change the colors of Bar, Track and Scale of the Easing Pie Chart for Reviews according to your plan.

    Changing the Color of Earning (Easing Pie Chart): 
    Locate the code as shown below:

    Here you can change the colors of Bar, Track and Scale of the Easing Pie Chart for Earning according to your plan.

    3. How to add Scroll Bar to any Div or Grid?

    Go to the folder with the name of “css” which contains a file “style.css” and open it on your HTML editor. Locate the line No. 2660 where you will find the following code:

    #scrollbox9 {
    overflow: auto;
    height: 473px;
    padding: 0 5px;
}

    Copy this code and paste the same just beneath the existing code. Here, you are also required to give it a unique ID by replacing “#scrollbox9” with the new one, for example “#scrollbox10”, etc. Similarly, in the next step, customize the height and padding according to your requirement. In the third step, go to the Div or Grid where you want to apply the Scroll Bar and mention the ID of scroll bar therein, for example, <div id=”scrollbox10”>
